Gamifying Speech Therapy

Adventure Biofeedback

In a team of five, we built a suite of tools for Speech Language Pathologists to be used with their patients ages 3-7. The main purpose of the tools is to keep the patient engaged with their clinician by gamifying the session. The system also improves the ease of use for clinicians to drive the session, take notes, create templates containing speech exercises for their patients, and sharing those templates among other clinicians. I was a main part of the development team as a full-stack developer.

  • Solved issue where speech therapy patients, ages 3-7, were not engaged in their Zoom session with their clinician.
  • Enhanced interaction by creating a web application using React and Flask with a Python backend where patients are tasked with completing a world map where each level has X number of activities set by the clinician.
  • Led demos to several Speech Language Pathologists over the course of 10 weeks and was able to gather optimal feedback on how to increase the engagement with the children.
  • Created a NoSQL database using MongoDB that stores user information and templates that they created on the web-application.
  • Innovated a game flow where the clinician navigates their patient through a world by using a control panel that emits the current game state which is done with Socket.IO.

Information Retrieval
